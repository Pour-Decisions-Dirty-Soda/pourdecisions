# Pour Decisions Dirty Soda

A vibrant, bubbly, and light one-page website for the **Pour Decisions Dirty Soda** food popup business. This site is designed to showcase the brand, upcoming events in Davis County, and provide a placeholder for the menu and social media links.

## ✨ Features

-   **Bubbly Aesthetic:** A soft blue and yellow color palette with rounded, friendly typography.
-   **Interactive Background:** Dynamic, floating soda bubbles generated via JavaScript.
-   **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
-   **Events Carousel:** A horizontal scrollable section to keep track of upcoming popups and markets.
-   **Sections Included:**
    -   **Hero:** Impactful intro with call-to-action buttons.
    -   **Menu:** Organized categories for sodas and treats.
    -   **Events:** List of upcoming appearances (Davis County Liberty Days, Kaysville First Friday, etc.).
    -   **Contact:** Direct email link and social media placeholders.

## 🛠️ Tech Stack

-   **HTML5:** Semantic structure.
-   **CSS3:** Custom animations, Flexbox/Grid layouts, and bubbly styling.
-   **JavaScript (Vanilla):** Background bubble engine and smooth-scroll interactivity.
-   **Fonts:** [Fredoka](https://fonts.google.com/specimen/Fredoka) and [Quicksand](https://fonts.google.com/specimen/Quicksand) via Google Fonts.
